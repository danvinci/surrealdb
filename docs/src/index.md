# SurrealDB.jl

Julia client for [SurrealDB](https://surrealdb.com). Talks to a remote
`surreal` server over WebSocket or HTTP, or runs the database in-process
via `libsurreal`.

The README on the [GitHub repo](https://github.com/danvinci/surrealdb) is
the canonical introduction with worked examples for every major feature.
This site is the auto-generated API reference.

## Install (alpha)

```julia
using Pkg
Pkg.add(url="https://github.com/danvinci/surrealdb", rev="v0.1.0-alpha.1")
```
